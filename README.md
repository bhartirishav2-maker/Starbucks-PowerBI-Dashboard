# Starbucks Sales Dashboard — Power BI

## Overview
Interactive 3-page sales dashboard built in Power BI
using a 1000-row Starbucks transaction dataset.

## Pages
- **Page 1** — Revenue by City + Top 10 Products by Margin %
- **Page 2** — Sales Heatmap (Day × Daypart)
- **Page 3** — Loyalty Tier + Payment Method Analysis

## Dataset
- 1000 transactions
- 25 columns
- 10 stores across 5 US cities

## Tools Used
- Power BI Desktop
- CSV Dataset (custom generated)
